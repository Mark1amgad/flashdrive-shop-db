# Flash Drive Ordering Web App

A lightweight web application built to organize bulk flash drive orders for students at Don Bosco Institute. Users select flash drive capacity, enter contact details, and submit requests that are automatically stored in Firebase. Features include a responsive design for all devices and an admin panel for reviewing and exporting orders.

## Overview

Developed to streamline the process of collecting flash drive orders from Informatica students. The system automatically organizes submissions in Firebase, reducing manual errors and coordination time.

## Features

- Simple form for submitting flash drive requests
- Select flash drive capacity (8GB, 16GB, 32GB, etc.)
- User contact information collection
- Data automatically stored in Firebase Realtime Database
- Responsive design for desktop and mobile devices
- Admin panel to review and export orders
- Real-time order tracking

## Technologies Used

- **HTML/CSS/JavaScript** - Front-end interface
- **Firebase Hosting** - Web hosting
- **Firebase Realtime Database** - Data storage
- **Responsive Design** - Mobile-friendly layout

## Key Learnings

- Efficient Firebase integration for real-time data management
- Responsive design principles for cross-device compatibility
- Building lightweight, user-friendly interfaces for real-world student projects
- Streamlining order collection processes

## Project Context

This system successfully reduced coordination time and manual errors when organizing bulk flash drive purchases for students, demonstrating practical application of web development skills in solving real-world organizational challenges.
